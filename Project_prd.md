# Capy's Money PRD Summary

## 1. Product Vision
Capy's Money is a personal finance management app for Vietnamese users, focusing on "3-second" recording, budget limits, and the 6-jar reporting model.

## 2. Key Goals
- High daily engagement (≥55% active users recording daily).
- Budget discipline through alerts.
- Insightful 6-jar reports.
- Premium conversion (ARPU ≥ 12,000 VND/month).

## 3. Core Features
- **Onboarding**: Email/OTP, goal setting.
- **Fast Recording**: Floating "+" button, scan/voice, auto-suggest.
- **Budgets**: Category-based limits with 80%/100% warnings.
- **Reporting**: 6-jar dashboard, PDF/Excel export.
- **Multi-wallet**: Cash, bank, and shared group wallets.
- **Premium**: Automation rules, unlimited sync, ad-free.

## 4. Design Direction (per User Reference)
- **Style**: Soft, playful, and modern.
- **Palette**: Pastel pinks, soft corals, and clean whites.
- **Elements**: Rounded corners, card-based layouts, and cute iconography.
